<caption></capiton>
 Essa tag define uma legenda de tabela HTML.

Essa tag deve ser inserida imediatamente após a tag <caption></capiton>.

Por padrão, ma legenda de tabela fica alinhada ao centro acima da tabela HTML.
No entanto, as propriedades do CSS text-align e caption-site podem ser usadas ára alinhar e colocar a legenda.

<thead></thead>
 Essa tag é usada para agrupar o conteúdo de cabeçalho em uma tabela HTML.

<tbody></body>
 Essa tag é usada para agrupar o conteúdo do corpo em uma tabela HTML.

Ela deve ser filha do elemento <table></table>, após qualquer elemento <caption></caption>, <colorgroup></colorgroup> e <thead></thead>.

<tfoot></tfoot>
 Essa tag é usada para agrupar o conteúdo de rodapé de uma tabela HTML.

Ela deve ser filha do elemento <table></table>, após qualquer elemento <caption></caption>, <colorgroup></colorgroup>, <thead></thead> e <tbody></body>.

Dica:
 Essas três tags não vão afetar o layout da tabela por padrão.
No entanto, pode ser estilizar esses elementos com CSS.
